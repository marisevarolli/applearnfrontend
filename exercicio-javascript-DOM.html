O que o JavaScript pode fazer?

JavaScript pode modificar elementos hmtl. Como pode serem feitas essas modificações?

    1. Pode realizar alterações no conteúdo do elemento HTML
    2. Pode realizar alterações nos atributos do elemento HTML.
    3. Pode realizar alterações nos estilos dos elementos HTML.
    4. Pode esconder ou mostrar o elemento HTML.
    5. Pode criar ou excluir um elemento HTML.
